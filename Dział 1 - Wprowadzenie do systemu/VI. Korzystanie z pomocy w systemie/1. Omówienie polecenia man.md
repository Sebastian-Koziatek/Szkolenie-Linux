Polecenie man to narzędzie służące do wyświetlania stron pomocy, stosowane w systemach Unix i uniksopodobnych. Nazwa jest skrótem od ang.manual–instrukcja obsługi.
Strony podręcznika są podzielone na kilka sekcji w oparciu o obszary, które obejmują:

| Numer sekcji |              Kategoria              |
|:------------:|:-----------------------------------:|
|       1      |            komendy użytkownika      |
|       2      |         Wywołania systemowe         |
|       3      |         Funkcje bilbioteki C        |
|       4      |           Pliki specjalne           |
|       5      |            Formaty plików           |
|       6      | Gry komputerowe i wygaszacze ekranu |
|       7      |                Różne                |
|       8      |   Polecenia systemowe i daemony     |

___
***Przykłady użycia:***

Jeżeli nie wiesz co wykonuje dana komenda, to o ile program posiada strony man możesz podejrzeć jej opis:

```
man -f komenda
```

np:
```
man -f date
man -f ls
man -f pwd
man -f passwd
itd
```
